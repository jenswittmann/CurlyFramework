Text to Speech

When it comes to screen readers, programmes such as NVDA and JAWS tend to spring to mind.

However, there are also cases where simply reading a main text aloud is useful.

For example, this could be useful when reading long texts is too tiring for the eyes, or when on parental leave and both hands are needed to care for a baby.

In these cases, it can be helpful to read a text aloud without having to install a programme beforehand.

The Web Speech API already offers a native option for this in the browser. The script is pure vanilla JavaScript based on AlpineJS and is quite simple — it is designed to serve as inspiration.

Use the left and right arrow keys to navigate. Press Escape to cancel.

Zoom Detector

Zoom: %

Positive:

Fixed or sticky elements can sometimes use a large amount of screen space, especially when the user zooms in to more than 150%. In this case, it may be better to inherit these elements.

In this script, you divide the outerWith width by the innerWidth to calculate the zoom factor. If the zoom factor is greater than 100%, you can add a class to reset the position. Try this with a Chromium browser using the Ctrl and +/- keys. It's actually a proof of concept, so the CSS isn't in the core at the moment.
