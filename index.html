Better accessibility and sustainability in web development?

It doesn't have to be a complete overhaul, small steps can help you get there. I recommend Michelle Barker's talk "Building a greener web" at "All Day Hey! 2023". Here are some tips:

  • Use WebP/AVIF and native lazyload (images, embeds).
  • Use fewer fonts and try to use variable fonts (.woff2).
  • Load embeds only on user interaction.
  • Test keyboard navigation and reduced-motion.
  • Find a community to test and discuss accessibility and sustainability with real people.

This toolset is designed to inspire you and provide you with learning resources. Feel free to copy, use, share and collaborate with it.

Get it now:

$ npm i curlyframework

Include the CSS and JS files in your markup and you are ready to go! Override the CSS settings using custom properties and use inline JS via AlpineJS to skip the build process. Need more? Install dependency via npm i and use CodeKit (or use the fallback npm commands) to compile your assets. Play around, learn and share.
