CurlyFramework

Persönliches Frontend-Framework für barrierearme Webprojekte.

$ npm i curlyframework & Github

Beispiele

Das Script kümmert sich auch um den Focus, damit dieser im geöffneten Fenster bleibt. Hier ein Beispiellink zum Test.

Alle Elemente skalieren auf Wunsch über die Klasse .scale-content im HTML Knoten proportional mit. Zusammen mit dem CSS Grid und den Klassen für die responsive Schrift eine super Basis für Designvorgaben aus Adobe XD und zeplin.io.

Die .htaccess wurde um Caching- und Security-Einstellungen ergänzt. Zusammen mit ContentBlocks von modmore, lässt sich ein flexibler Webseiten-Baukasten mit 100% Code-Kontrolle umsetzen. Die Datei modx.css bringt ein minimalistisches Layout für das Backend mit.
