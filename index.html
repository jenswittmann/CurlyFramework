CurlyFramework

Persönliches Frontend-Framework für barrierearme Webprojekte.

$ npm i curlyframework & Github

Beispiele

Die Javascript Klasse a11y.js kümmert sich um die Verwaltung der ARIA Attribute. Damit können einfache barrierearme Overlays erstellt werden.

Das Script kümmert sich auch um den Focus, damit dieser im geöffneten Fenster bleibt. Hier ein Beispiellink zum Test.

Über die ARIA Attribute kann ebenfalls ein Akkordeon umgesetzt werden.

Alle Elemente skalieren auf Wunsch über die Klasse .scale-content im HTML Knoten proportional mit. Zusammen mit dem CSS Grid und den Klassen für die responsive Schrift eine super Basis für Designvorgaben aus Adobe XD und zeplin.io.

Ein einfaches responsives Raster auf Basis von Flexbox und CSS Grid ist auch enthalten. Die Spaltenanzahl kann über $grid-cols und der Zwischenabstand über $grid-gap angepasst werden.

MODX ist eins der flexibelsten Systeme – für den schnellen Start sind die Dateien .gitignore und .gitify mit dabei. Die .htaccess wurde um Caching- und Security-Einstellungen ergänzt. Zusammen mit ContentBlocks von modmore, lässt sich ein flexibler Webseiten-Baukasten mit 100% Code-Kontrolle umsetzen. Die Datei modx.css bringt ein minimalistisches Layout für das Backend mit.

Die Grundlage bildet Tachyons als modulares CSS Framework.
